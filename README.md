# E-Commerce Website Using Flask

## Project Overview

A full-stack E-Commerce web application built using Flask, SQLAlchemy, Bootstrap, and SQLite.

## Features

* User Registration
* User Login & Logout
* Product Catalog
* Product Search
* Product Categories
* Product Images
* Shopping Cart
* User-Specific Cart
* Wishlist
* Product Reviews & Ratings
* Checkout System
* Orders Page
* Admin Dashboard
* Add Product
* Edit Product
* Delete Product

## Technologies Used

* Python
* Flask
* SQLAlchemy
* SQLite
* Bootstrap 5
* HTML
* CSS
* Jinja2

## Installation

```bash
py -3.10 -m pip install -r requirements.txt
```

## Run Project

```bash
py -3.10 app.py
```

Open:

http://127.0.0.1:5000
